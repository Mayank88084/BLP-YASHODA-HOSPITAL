# BLP YASHODA Hospital Website

This is a simple multi-page hospital website built with HTML, CSS, and JavaScript.

## Files

- `index.html` - Home page
- `about.html` - About page
- `doctors.html` - Doctors page
- `contact.html` - Contact page
- `style.css` - Shared design and layout
- `script.js` - Menu and form behavior
- `images/` - Doctor images

## How to Upload to GitHub Without Git Installed

1. Go to [GitHub](https://github.com/) and sign in.
2. Click `New repository`.
3. Repository name suggestion: `blp-yashoda-hospital-website`
4. Keep it `Public`.
5. Click `Create repository`.
6. On the next page, choose `uploading an existing file`.
7. Drag and drop all project files and the `images` folder contents.
8. Click `Commit changes`.

## How to Turn It Into a Live Website With GitHub Pages

1. Open your repository on GitHub.
2. Click `Settings`.
3. Click `Pages`.
4. Under `Build and deployment`, select:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
5. Click `Save`.
6. Wait for GitHub Pages to publish the site.
7. Your website link will look similar to:
   `https://your-github-username.github.io/blp-yashoda-hospital-website/`
